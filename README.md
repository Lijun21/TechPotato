# TechPotato
nodejs, express, mongoDB

# www.tPotato.com

## Implementation

Install MongoDB  

Learn more at  [MongoDB](https://docs.mongodb.com/manual/tutorial/install-mongodb-on-os-x/)

start mongo server

```
mongod
```

install dependencies

```
npm install
```

start the server

```
npm start
```
